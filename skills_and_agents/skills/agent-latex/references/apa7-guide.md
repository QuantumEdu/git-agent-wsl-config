# Guía APA 7ma Edición - Referencias y Formato

## Cambios Principales APA 7ma Edición

### Referencias con Hanging Indent

- Todas las referencias usan **hanging indent** (sangría francesa)
- Primera línea al margen izquierdo, líneas subsiguientes indentadas 0.5 pulgadas
- Lista alfabética por apellido del primer autor

### Formato de Autor

| N.$^o$ autores | Formato |
|----------------|---------|
| 1 | Apellido, A. A. |
| 2 | Apellido, A. A., \& Apellido, B. B. |
| 3-20 | Apellido, A. A., Apellido, B. B., \& Apellido, C. C. |
| 21+ | Apellido, A. A., Apellido, B. B., ... Apellido, Z. Z. |

### Formato de Año

- Año en negrita en referencias
- "En prensa" para documentos aceptados pero no publicados
- "En preparación" para manuscritos en preparación
- "Manuscrito no publicado" para trabajos no aceptados

## Estructura del Documento APA 7ma

### Página de Título

Elementos requeridos:
1. Título del trabajo (centrado, en mayúscula solo primera letra y nombres propios)
2. Nombre del autor(es)
3. Afiliación (universidad, departamento)
4. Nota de autor (opcional)
5. Pie de página con autor de correspondencia

### Abstract

- Una página después de la página de título
- Centrado, en negrita, sin cursiva: "Abstract"
- 150-250 palabras
- Lista de palabras clave debajo (indentadas)

### Body

- Comienza en nueva página
- Títulos centrados en negrita
- Primera letra de título en mayúscula
- Sin numeración de sección

### Referencias

- Nueva página al final
- Título "Referencias" centrado y en negrita
- Hanging indent de 0.5 pulgadas

## Formato de Referencias - Ejemplos

### Artículo de Revista

```
Autor, A. A., \& Autor, B. B. (Año). Título del artículo.
Revista, volumen(número), páginas. https://doi.org/xxxxx
```

**Ejemplo:**
```
García, M. A., \& López, C. R. (2024). Efecto del estrés en el rendimiento.
Revista de Psicología, 12(3), 45-58. https://doi.org/10.1234/rp.2024.001
```

### Libro

```
Autor, A. A. (Año). Título del libro en cursiva (Edición). Editorial.
```

**Ejemplo:**
```
Hernández, R. (2023). Metodología de la investigación (6.$^a$ ed.). McGraw-Hill.
```

### Capítulo de Libro

```
Autor, A. A. (Año). Título del capítulo. En A. A. Autor (Ed.),
Título del libro en cursiva (pp. xx-xx). Editorial.
```

### Tesis o Disertación

```
Autor, A. A. (Año). Título de la tesis en cursiva [Tesis doctoral,
Universidad]. Nombre del repositorio. URL
```

### Página Web

```
Autor, A. A. (Año, día de mes). Título del artículo. Nombre del sitio web.
URL
```

### Video

```
Autor, A. A. [Canal]. (Año, día de mes). Título del video [Video]. YouTube.
URL
```

## Citas en Texto

### Cita directa corta (menos de 40 palabras)

```
"García (2024) encontró que..." o "Los resultados indican que... (García, 2024)"
```

### Cita directa larga (40+ palabras)

- Sangría de 0.5 pulgadas en todo el bloque
- Sin comillas
- Mismo formato que el texto

### Cita paráfrasis

```
(García, 2024) - parenthetical
García (2024) encontró que... - narrative

```

### Citas múltiples

```
(García, 2024; López, 2023) - orden alfabético
(García, 2024a, 2024b) - mismo autor, diferentes trabajos
```

## Tablas y Figuras

### Tablas

- Caption centrada arriba de la tabla
- En negrita: "Tabla 1" seguido del título
- Notas debajo de la tabla

```
\begin{table}[htbp]
\centering
\caption{Resultados del Análisis}
\begin{tabular}{lcc}
\toprule
Variable & Media & DE \\
\midrule
X & 10.5 & 0.3 \\
\bottomrule
\end{tabular}
\end{table}
```

### Figuras

- Caption centrada debajo de la figura
- En negrita: "Figura 1" seguido del título
- Reproducción de materiales ajenos requiere permiso

## Formato de Párrafos

- Sangría de primera línea: 0.5 pulgadas
- Times New Roman 12pt
- Doble espacio
- Márgenes: 1 pulgada todos los lados
- Justificado

## DOI y URLs

- DOI formateados como URL: https://doi.org/xxxxx
- NO usar "Recuperado de" o "Disponible en"
- Solo incluir el DOI o URL directo

## Mensajes de Error Comunes

1. **Falta año**: Todos los recursos deben tener año
2. **Capitalización incorrecta**: Solo primera letra y nombres propios en títulos
3. **DOI faltante**: Incluir siempre que esté disponible
4. **Hanging indent incorrecto**: Verificar sangría de 0.5 pulgadas
5. **Orden alfabético incorrecto**: Verificar orden de autores
