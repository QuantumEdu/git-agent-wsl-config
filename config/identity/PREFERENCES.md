# PREFERENCES.md - Stack y Convenciones Generales

## Stack Principal
- Python (FastAPI, pytest, poetry)
- Rust (Cargo)
- SQLite (Engram / FTS5)
- Docker para despliegues

## Convenciones de Código
- En Python usar anotaciones de tipo (`type hints`).
- Seguir la arquitectura hexagonal o limpia siempre que sea viable en proyectos de backend.
- Prefiere modularidad y claridad ante optimizaciones prematuras.
